Website Planning Document

Project for WDD131 – Dynamic Web Fundamentals

Site Name

Example Site Name: Healthy Living Hub

I chose this name because the site will focus on simple, practical health and wellness tips, helping users learn about nutrition, mental health, physical activity, and lifestyle habits. It reflects positivity and personal improvement.

Site Purpose

The purpose of this website is to provide users with helpful health resources, including fitness guides, nutrition suggestions, wellness articles, and a form that allows users to subscribe to personalized wellness updates. The site will be informative, engaging, and supportive for anyone looking to improve their daily lifestyle.

Scenarios

  • “What are some simple daily habits I can follow to improve my overall wellness?”
  • “How can I create a balanced meal plan without spending too much money?”

Color Scheme

Primary Color – Deep Blue (#004AAD): Used for headings, navigation, and important accents.

Secondary Color – Warm Orange (#F5A623): Used for call-to-action buttons and highlights.

These colors are applied directly in this document as required by the rubric.

Typography

Font Selected: Poppins

  • Headings: Poppins bold (700)
  • Body text: Poppins regular (300–500)

This document uses the same typography as the final project will use.
